LE JEU DU SERPENT

Voici une version du jeu du serpent codée en Javascript

Commandes du Jeu

Touches directionnelles : pour contrôler le serpent

Barre d'espace : pour recommencer

Touche P : pour mettre en pause et sortir de pause

Touche CTRL : pour un boost de 3 secondes

BOOST : une charge toutes les 5 pommes mangées

Bullet Time : remplace le boost à partir de 20 pommes mangées...

Choississez votre mode de difficulté de départ
